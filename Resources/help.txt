Список команд:
iterations # - установка количества повторений (#) для вычисления среднего времени работы
               количество повторений по умолчанию: 100
               пример: iterations 100           
sequence #   - установка последовательности для тестирования
               пример: sequence 5 4 9 7 5 6
random #     - генерация случайной последовательности для тестирования, где # - длина последовательности
               данная команда может быть использована вместо команды "sequence"
test -  запуск работы алгоритмов с установленными параметрами
exit - завершение работы с программой
